  • IMPORTANT UPDATE:
    tl;dr: If you want the best privacy, I recommend choosing the "Block and Re-load" Stripping Method and enabling the "Skip Known Redirects" checkbox.

    One recent addition to the extension that's quite important is the ability to "Skip Known Redirects." If enabled, this feature will recognize certain links that will pass you through some "intermediate" URL before being redirected to the final URL that you're after, call the "target". An example of this would be Google Search result links where you are briefly passed through a Google tracking page before being redirected to the target URL of the search result you clicked. Some affiliate links also do this same type of thing. These links all have some things in common:

    1. The intermediate URL that is visited is not the target URL that you are ultimately going to land on.
    2. The intermediate URL is usually there only to cookie/track you.
    3. The actual target URL is embedded somewhere in the parameters of the intermediate URL, so we can grab it and skip the intermediate step.
